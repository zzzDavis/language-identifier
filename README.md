# Multilingual Language Identification

## Overview

This project is a machine learning application that classifies five-letter words as English, German, or Spanish. The classifier compares multiple machine learning algorithms to evaluate their effectiveness for short-text language identification.

The project was developed as part of graduate coursework in Machine Learning and Natural Language Processing.

---

## Features

- Language classification for English, German, and Spanish
- Automatic text normalization
- Removal of duplicate words across language datasets
- Balanced dataset creation
- Character n-gram TF-IDF vectorization
- Positional character feature engineering
- Performance comparison between:
  - Support Vector Machine (SVM)
  - K-Nearest Neighbors (KNN)
  - Multi-Layer Perceptron (MLP)
- Confusion matrix generation
- Accuracy visualization with Matplotlib
